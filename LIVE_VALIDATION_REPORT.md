## 2. Failures Log & Fixes

| Issue | Root Cause | Fix |
|-------|------------|-----|
| FastEmbed BM25 segfault | Python 3.14 + ONNX runtime incompatibility | Implemented local BM25 sparse vector generator in `sparse_vector.py` |
| SparseIndexParams error | Invalid `on=True` param in qdrant-client v1.12.x | Changed to `on_disk=False` |
| Alembic hangs in async | `command.upgrade()` calls `asyncio.run()` internally | Use `subprocess.run()` to run Alembic in separate process |
| DoclingDocument.title missing | API change in Docling version | Extract title from first markdown heading instead |
| Storage path conflict | Derived assets stored under file path | Store in `_derived/` sibling directory |
| get_sentence_embedding_dimension deprecated | API change in sentence-transformers | Use `get_embedding_dimension()` |
